# Electricity

電力數據分析專案，用於分析電力使用數據、用電量統計與趨勢視覺化。

## Getting Started

### Prerequisites

- Python 3.10+
- pip

### Installation

```bash
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```
